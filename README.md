# OCEANWAVE CORE

Core Modules for OCEANWAVE Apps.

## Install
```bash
npm i oceanwave@core
```
## Examples
Create an OCEANWAVE App:
```js
import {core} from "@oceanwave/core";
//OCEANWAVE App Entry
core.app.main(()=>{
  core.log.info("Hello world")
  return 0;
})
```
